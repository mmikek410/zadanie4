Инструкция

есть проект java, где есть следующие классы cards, JsonSimpleParser, Main, Root

а так же файл test.json откуда мы берем json данные для парсинга(данные о владельцах карт), который просиходит внутри класса JsonSimpleParcer. 

Папку untitled, надо открыть с помощью программы Intellij idea, данная программа парсит json с помощью библиотеки от google json-simple в фреймворке Maven.

По итогу программа выводит название файла и значения объектов json в виде строки.
